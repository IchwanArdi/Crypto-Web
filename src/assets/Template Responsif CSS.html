/* Default Styles (Mobile First) */
body {
font-family: Arial, sans-serif;
margin: 0;
padding: 0;
box-sizing: border-box;
}

.container {
width: 90%;
margin: auto;

}

/* Small devices (phones) */
@media (max-width: 600px) {
.container {
background: lightblue;
}
}

/* Medium devices (tablets) */
@media (min-width: 601px) and (max-width: 1024px) {
.container {
background: lightgreen;
}
}

/* Large devices (laptop, desktop) */
@media (min-width: 1025px) and (max-width: 1440px) {
.container {
background: lightcoral;
width: 80%;
}
}

ChatGPT bilang:

Kode CSS yang kamu tulis memiliki dua ::after pseudo-element pada .HeroSection, tetapi hanya yang terakhir yang akan diterapkan karena elemen ::after hanya bisa muncul sekali dalam satu elemen.

Untuk mengatasi ini, gunakan ::before untuk gradient di bagian atas dan ::after untuk gradient di bagian bawah. Berikut adalah versi yang diperbaiki:

css
.HeroSection {
  position: relative;
  width: 100%;
}

/* Gradient atas */
.HeroSection::before {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 30%; /* Bisa disesuaikan */
  background: linear-gradient(to top, rgba(0, 0, 0, 0) 0%, black 100%);
  pointer-events: none; /* Supaya tidak mengganggu interaksi */
}

/* Gradient bawah */
.HeroSection::after {
  content: '';
  position: absolute;
  bottom: 0;
  left: 0;
  width: 100%;
  height: 10%; /* Bisa disesuaikan */
  background: linear-gradient(to bottom, rgba(0, 0, 0, 0) 0%, black 100%);
  pointer-events: none; /* Supaya tidak mengganggu interaksi */
}

Dengan cara ini, .HeroSection akan memiliki gradient di bagian atas dan bawah secara bersamaan. 🚀
